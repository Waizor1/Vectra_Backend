## Гайд по админке Directus (TVPN)

### 0) Режим гайда для команды (шаги + скриншоты)
1. **Навигация** — слева основные разделы и папки. Ориентир: папка «Основное».
   Скриншот: `assets/guide-navigation.png`
2. **Главный дашборд** — ключевые метрики и графики.
   Скриншот: `assets/guide-dashboard.png`
3. **Список пользователей** — фильтры, поиск, статусы, быстрый доступ к карточке.
   Скриншот: `assets/guide-users-list.png`
4. **Карточка пользователя** — вкладки и подсказки, где менять лимиты/подписку.
   Скриншот: `assets/guide-user-edit.png`
5. **Промо и призы** — см. разделы Promo Codes / Prize Wheel для типовых операций.

### 1) Куда заходить и что смотреть
- **Users** — основная карточка пользователя: подписка, лимиты, баланс, блокировки.
- **Active Tariffs** — текущие активные тарифы (LTE‑лимиты, мультипликаторы, usage).
- **Tariffs** — справочник тарифов и цен.
- **Promo Codes / Promo Batches / Promo Usages** — промокоды, их партии и история использования.
- **Prize Wheel Config / Prize Wheel History** — настройки «колеса» и история выпадений.
- **Connections** — данные для графиков подключений.
- **Processed Payments** — обработанные платежи.

### 2) Частые действия (как работать)
**Поменять LTE лимит пользователю**
1. Открыть Users → найти пользователя.
2. Изменить `lte_gb_total`.
3. Сохранить — Directus вызовет backend для синхронизации.

**Продлить/ограничить подписку**
1. Users → `expired_at` или `hwid_limit`.
2. Сохранить — backend синхронизирует RemnaWave.

**Обновить LTE лимит активного тарифа**
1. Active Tariffs → `lte_gb_total`.
2. Сохранить — backend синхронизирует LTE.

**Промокод**
1. Promo Codes → создать запись.
2. В `code_hmac` можно указать «сырой» код — хук преобразует в HMAC.

**Колесо призов**
1. Prize Wheel Config → изменить вероятность или приз.
2. Сумма вероятностей активных призов должна быть ≤ 1.0.
3. Для `subscription` нужно указывать целое число дней в `prize_value`.

### 3) Дашборды и метрики
Раздел **Аналитика** (`/admin/insights`) — список дашбордов и панелей.
API для графиков: `/admin-widgets/*`
- `total-users`
- `active-users`
- `inactive-users`
- `registered-users`
- `connections`

### 4) Роли (команда)
- **Administrator** — полный доступ.
- **Manager** — основной CRUD (users/active_tariffs/tariffs/promo/prize_wheel).
- **Viewer** — только чтение.

- Некоторые служебные таблицы помечены как «Служебное».
- Если что‑то не отображается, проверь права/политику.
