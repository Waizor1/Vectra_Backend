# Система прогрессивного ценообразования тарифов

## Обзор

Система тарифов переработана для поддержки **прогрессивной скидки**, где чем больше устройств покупает пользователь, тем выгоднее становится каждое следующее устройство. Это мотивирует пользователей покупать больше устройств.

## Принцип работы

### Формула расчета

Каждое следующее устройство стоит дешевле предыдущего по экспоненциальной формуле:

```
Общая цена = базовая_цена + Σ(базовая_цена × (множитель ^ (номер_устройства - 1)))
```

Где:
- **базовая_цена** - цена за первое устройство
- **множитель** - коэффициент прогрессивной скидки (0.1-1.0)
- **номер_устройства** - порядковый номер устройства (2, 3, 4, ...)

### Примеры расчета

**Тариф с base_price=100₽, progressive_multiplier=0.9:**

| Устройств | Расчет | Итого | Средняя цена | Экономия |
|-----------|--------|-------|--------------|----------|
| 1 | 100₽ | **100₽** | 100₽ | 0% |
| 2 | 100₽ + 100₽×0.9 | **190₽** | 95₽ | 5% (10₽) |
| 3 | 100₽ + 90₽ + 81₽ | **271₽** | 90₽ | 10% (29₽) |
| 5 | 100₽ + 90₽ + 81₽ + 73₽ + 66₽ | **410₽** | 82₽ | 18% (90₽) |
| 10 | 100₽ + 90₽ + ... + 39₽ | **651₽** | 65₽ | 35% (349₽) |

**Тариф с base_price=100₽, progressive_multiplier=0.8 (более агрессивная скидка):**

| Устройств | Расчет | Итого | Средняя цена | Экономия |
|-----------|--------|-------|--------------|----------|
| 1 | 100₽ | **100₽** | 100₽ | 0% |
| 2 | 100₽ + 100₽×0.8 | **180₽** | 90₽ | 10% (20₽) |
| 3 | 100₽ + 80₽ + 64₽ | **244₽** | 81₽ | 19% (56₽) |
| 5 | 100₽ + 80₽ + 64₽ + 51₽ + 41₽ | **336₽** | 67₽ | 33% (164₽) |

## Изменения в структуре данных

### Модель Tariffs

**Новые поля:**
- `progressive_multiplier` (REAL) - множитель прогрессивной скидки (0.1-1.0)

**Удаленные поля:**
- `device_discount` - заменен на `progressive_multiplier`
- `hwid_limit` - больше не нужен, так как лимит определяется выбором пользователя

## Настройки множителя

### Рекомендуемые значения

- **0.95** - минимальная скидка (5% на каждое устройство)
- **0.9** - умеренная скидка (10% прогрессия) 
- **0.85** - средняя скидка (15% прогрессия)
- **0.8** - высокая скидка (20% прогрессия)
- **0.7** - максимальная скидка (30% прогрессия)

### Влияние на экономию (для 5 устройств, base_price=100₽)

| Множитель | Итого | Экономия | Средняя цена |
|-----------|-------|----------|--------------|
| 0.95 | 477₽ | 4.6% (23₽) | 95₽ |
| 0.9 | 410₽ | 18% (90₽) | 82₽ |
| 0.85 | 352₽ | 30% (148₽) | 70₽ |
| 0.8 | 336₽ | 33% (164₽) | 67₽ |
| 0.7 | 240₽ | 52% (260₽) | 48₽ |

## Преимущества новой системы

### Для пользователей

1. **Реальная экономия** при покупке нескольких устройств
2. **Мотивация покупать больше** - каждое устройство дешевле
3. **Прозрачность** - видно точную экономию в рублях и процентах
4. **Гибкость** - можно выбрать любое количество устройств

### Для бизнеса

1. **Увеличение AOV** (Average Order Value)
2. **Стимулирование крупных покупок**
3. **Конкурентное преимущество**
4. **Простота настройки** - один параметр для управления скидками

### Для администраторов

1. **Простая настройка** - один множитель вместо сложных правил
2. **Гибкий контроль** - можно настроить агрессивность скидок
3. **Прогнозируемость** - понятная математическая модель
4. **Легкое тестирование** A/B разных множителей
