# ⌨️ Автоматическая установка админских клавиатур

Система автоматической установки и управления админскими клавиатурами для удобного доступа к админ-меню.

## 🚀 Автоматическая установка

### При старте бота (`/start`)
Админская клавиатура автоматически устанавливается при выполнении команды `/start`:

```python
# В bloobcat/bot/routes/start.py
user = await Users.get_user(message.from_user)
if user and user.is_admin:
    await set_admin_keyboard(message.bot, user.id)
```

**Как это работает:**
1. Пользователь вводит `/start`
2. Система проверяет статус `is_admin` в БД
3. Если пользователь админ → автоматически устанавливается клавиатура
4. Админ сразу видит кнопки "🔧 Админ панель" и "🧪 Тесты"

### При назначении/снятии прав
Клавиатура автоматически управляется при получении/потере прав администратора:

- Когда пользователь получает права админа в БД → клавиатура устанавливается при следующем `/start`
- При ручной установке через команды управления клавиатурами

## 🛠️ Команды управления

### Основные команды
```bash
/admin                      # Главное админ меню + установка клавиатуры
```

### Отладочные команды
```bash
/setup_admin_keyboard           # Установить клавиатуру себе
/setup_keyboard_for_user [id]   # Установить клавиатуру пользователю
/setup_all_admin_keyboards      # Установить всем админам
/remove_admin_keyboard          # Убрать клавиатуру
```

### Обычные пользователи

```

### Админы дополнительно видят
```

## ⚙️ Технические функции

### Установка клавиатуры
```python
from bloobcat.bot.routes.admin.admin_menu import set_admin_keyboard

await set_admin_keyboard(bot, user_id)
```

### Универсальная установка с проверкой
```python
from bloobcat.bot.routes.admin.admin_menu import setup_admin_keyboard_for_user

success = await setup_admin_keyboard_for_user(bot, user_id)
# Автоматически проверяет is_admin в БД
```

### Удаление клавиатуры
```python
from bloobcat.bot.routes.admin.admin_menu import remove_admin_keyboard_for_user

success = await remove_admin_keyboard_for_user(bot, user_id)
```

### Массовая установка
```python
from bloobcat.bot.routes.admin.admin_menu import setup_keyboards_for_all_admins

success_count, total_count = await setup_keyboards_for_all_admins(bot)
```

## 🔄 Сценарии использования

### 1. Новый админ
```bash
# 1. Назначаем права через БД или другие инструменты
# 2. Пользователь вводит /start
# 3. Клавиатура устанавливается автоматически
# 4. У пользователя появляются кнопки админ-панели
```

### 2. Снятие прав
```bash
# 1. Убираем права через БД или другие инструменты
# 2. Ручное удаление клавиатуры при необходимости:
/remove_admin_keyboard
```

### 3. Перезапуск бота
```bash
# Установить клавиатуры всем админам после перезапуска
/setup_all_admin_keyboards
```

### 4. Проблемы с клавиатурой
```bash
# Принудительно переустановить клавиатуру
/setup_admin_keyboard

# Убрать клавиатуру (для отладки)
/remove_admin_keyboard

# Восстановить клавиатуру
/admin
```

## 🔍 Логирование

Все операции с клавиатурами логируются:

```python
# Автоустановка при старте
logger.info(f"Автоматически установлена админская клавиатура для пользователя {user_id}")

# Назначение прав
logger.info(f"Администратор {admin_id} изменил статус админа пользователя {user_id}")

# Массовая установка
logger.info(f"Установлены клавиатуры для {success_count}/{total_count} админов")
```

## ⚠️ Особенности и ограничения

### Безопасность
- Админ не может снять права у самого себя через `/set_admin`
- Все операции доступны только админам (`IsAdmin()` фильтр)
- Клавиатура устанавливается только при наличии флага `is_admin=True` в БД

### Обработка ошибок
- Если пользователь заблокировал бота → ошибка логируется, но не прерывает работу
- При массовой установке ошибки для отдельных пользователей не прерывают процесс
- Неудачные установки отслеживаются и сообщаются

### Совместимость
- Работает параллельно с существующей кнопкой WebApp
- Не конфликтует с обычными пользователями
- Клавиатура `persistent=True` → сохраняется между сессиями
