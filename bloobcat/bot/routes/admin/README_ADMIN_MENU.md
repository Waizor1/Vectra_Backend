# 🔧 Новое Админ Меню

Централизованная система управления ботом с удобной навигацией через инлайн кнопки.

## 🚀 Быстрый старт

1. **Команда `/admin`** - устанавливает админскую клавиатуру и показывает главное меню
2. **Кнопка "🔧 Админ панель"** - быстрый доступ к главному меню
3. **Кнопка "🧪 Тесты"** - доступ к тестовым функциям

## 📱 Структура меню

### 🔧 Главное админ меню
```
👥 Управление пользователями  📊 Статистика
⚙️ Системные операции         📢 Рассылки  
🛠️ Утилиты
```

### 👥 Управление пользователями
- **📊 Общая статистика** - статистика по всем пользователям
- **🔍 Управление пользователем** - поиск и детальное управление
- **🚫 Заблокировавшие бота** - управление пользователями что заблокировали бота

### 🔍 Управление конкретным пользователем
После поиска пользователя доступны следующие функции:
- **📊 Информация** - детальная информация о пользователе
- **💰 Подписка** - управление подпиской и регистрацией
- **🔄 Автопродление** - настройка автопродления
- **🆓 Trial** - управление trial периодом
- **🚫 Статус блокировки** - управление флагом блокировки бота
- **🗑️ Удалить** - полное удаление пользователя

### 📊 Статистика и аналитика
- **🎯 UTM статистика** - общая и по конкретным UTM
- **📅 Временная статистика** - за день/неделю/месяц
- **🌐 Онлайн пользователи** - текущие активные пользователи
- **🎪 Trial статистика** - статистика trial extensions

### ⚙️ Системные операции
- **🔄 Проверка подписок** - ручной запуск проверки подписок
- **🆓 Проверка trial** - проверка пользователей с trial
- **🔄 Все проверки** - запуск всех системных проверок
- **⏰ Истекающие подписки** - список истекающих подписок

### 🧪 Тестовое меню
- **📢 Тестирование уведомлений** - тест всех типов уведомлений
- **💳 Тестирование платежей** - тест автоплатежей и уведомлений
- **📊 Тестирование статистики** - отправка тестовой статистики

## 🎯 Ключевые особенности

### ✅ Преимущества нового меню
- **Удобная навигация** - все функции в одном месте
- **Breadcrumb навигация** - всегда понятно где находишься
- **Быстрый доступ** - reply кнопки для входа в меню
- **Логическая группировка** - функции разбиты по категориям
- **Интерактивность** - кнопки вместо ввода команд
- **Совместимость** - все старые команды продолжают работать

### 🔄 Автоматическая установка клавиатуры
Админская клавиатура устанавливается автоматически при:
- Выполнении команды `/admin`
- Получении админских прав

### 📱 Reply клавиатура для админов
```
[🔧 Админ панель]  [🧪 Тесты]
```

## 🛠️ Технические детали

```

├── admin_menu.py          # Основное меню и reply клавиатура
├── keyboards.py           # Все инлайн клавиатуры
├── navigation.py          # Обработчики навигации

```

### Callback данные
```python
# Основная навигация
"admin:main"              # Главное меню
"admin:users"             # Управление пользователями
"admin:stats"             # Статистика
"admin:system"            # Системные операции

# Тестовое меню
"test:main"               # Главное тестовое меню
"test:notifications"      # Тест уведомлений
"test:payments"           # Тест платежей

# Управление пользователем
"admin:user:123:info"     # Информация о пользователе 123
"admin:user:123:trial"    # Trial управление для пользователя 123
```

## 🔧 Команды для управления

### Основные команды (работают как раньше)
```bash
/admin                    # Главное админ меню
/admin_help              # Справка по командам
/remove_admin_keyboard   # Убрать админскую клавиатуру (отладка)
```

### Управление пользователями
```bash
/set_registered 123 1     # Активировать пользователя
/set_trial 123 7         # Дать trial на 7 дней
/balance 123 +500        # Добавить 500₽ на баланс
/days 123 +30           # Добавить 30 дней подписки
/trigger_autopay 123     # Запустить автоплатеж (тест)
```

### Статистика
```bash
/stats                   # UTM статистика с пагинацией
/stat utm_name          # Статистика по конкретному UTM
/online                 # Онлайн пользователи
```

## 🔍 Поиск пользователей

Поддерживается поиск по:
- **ID пользователя**: `123456789`
- **Username**: `@username` или `username`

## ⚠️ Совместимость

### Старые команды
Все существующие команды продолжают работать:
- `/user_info 123` - информация о пользователе
- `/blocked_stats` - статистика заблокированных
- `/send` - массовая рассылка
- и другие...

### Миграция
- Новое меню работает параллельно со старыми командами
- Постепенно можно переходить на использование нового интерфейса
- Старые команды не удалены для совместимости
